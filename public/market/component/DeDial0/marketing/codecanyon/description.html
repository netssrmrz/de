DeDial is a simple User Interface component whose primary purpose is to draw a circular but digital dial. It's "digital" because individual "ticks" are drawn for each possible dial position, rather than a continuous circle.

These types of dials can be used to represent circular gauges, or the ticks on a clock dial, or segments of a targeting sight. The uses are ultimately up to your imagination.

DeDial can also be used as a single value counter. This is handy when you want to display an animated count of customers, orders, or cashflow, for example. Please note that, whilst this component can be used to build clock functionality, it does not include time-based counting as it is.

You can find additional information here. Reference documentation can be found here. Also note, this is not a Wordpress component.

Basic Usage

Usage is as easy as including the component file and an optional stylesheet. There are no other dependencies or files required. From there you can start placing DeDial tags anywhere in your HTML.

<link rel="stylesheet" href="./dedial.css">
<script src="dedial.js" type="module"></script>

<de-dial value="100" max-value="100"></de-dial>

Features

  • Great as a counter. Supports counting up and down.
  • Great for animations.
  • Ease to automate and script.
  • Fully customisable via CSS.
  • Based on SVG so rendering is always smooth and scalable.
  • Uses latest Web Components HTML standard so code will always be supported by browsers.
  • Easy to install with NO other dependencies, including jQuery.
  • Sample code and reference docs included.
  • Supports Chrome, Edge, Safari, Opera and Firefox.
  • Additional button component included so counters can be controlled by users.

Attributes

  • auto-start: Starts counting as soon as visible.
  • auto-stop: Stops counting when done.
  • count-reverse: Counts down instead of up.
  • gap-width: Set the width of the gap between ticks.
  • max-value: Set the maximum number of ticks.
  • show-label: Show counter number.
  • stop-href: URL to jump to when counting completes.
  • tick-width: Set the width of the ticks.
  • value: Set the starting or current value of the counter.
  • viewbox-radius: Facilitates large tick rendering.
  • wait-millis: Set the speed of counting.

Properties

  • value: Set the current counter value fro JavaScript.
  • labelText: Set the counter label from JavaScript.

Methods

  • reset(): Set the counter back to the starting value.
  • restart(): Reset the counter and start counting.
  • start(): Start counting.
  • stop(): Stop counting.
  • toggle(): Start or stop counting.

Events

  • completed: Fires when counting has completed.
  • tick: Fires on every count.
